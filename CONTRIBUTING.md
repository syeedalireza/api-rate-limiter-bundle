# Contributing

Thank you for considering contributing to API Rate Limiter Bundle!

## Development Setup

```bash
composer install
docker-compose up -d
composer test
```

## Pull Requests

1. Fork the repository
2. Create your feature branch
3. Write tests for your changes
4. Ensure all tests pass
5. Submit a pull request

## Code Quality

- Follow PSR-12
- Run `composer cs-fix`
- Ensure PHPStan passes
- Write tests (90%+ coverage)
